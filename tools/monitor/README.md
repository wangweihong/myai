# 监控
## 服务端
1. 下载脚本
2. 生成监控栈和granfa provisioning文件`python ai_monitor.py stack -t service`
3. 运行`python ai_monitor.py run -t service`
4. 访问localhost:3000, 默认账号密码admin/admin123

# agent

# 指令模板
## 人物图像分析
```
核心原则：请以绝对客观、中立的态度，扮演一个“人形扫描仪”的角色。您的描述应基于纯粹的可视事实，为图像生成一份结构化报告。避免一切主观评价、情感揣测和不必要的推断。

请严格按照以下结构与要素进行描述：

一、 全局元信息
图像类型：（例如：数码照片、艺术插画、影视截图、新闻图片）

核心主题：（用一句话概括图像的核心事件或焦点）

二、 人物综合描述（按图中出现顺序或显著度编号，如P1，P2）
对于图像中的每个核心人物，请按以下框架描述：

【P1：人物描述框架】

身份与位置：

身份/角色：（如可明确识别：厨师、跑步者；否则略过）

在画面中的位置：（如：位于前景中央、背景左侧三分之一处）

客观外貌特征：

年龄段：（如：青年、中年、老年，或估计年龄范围）

肤色：（使用通用描述，如浅肤色、中等肤色、深肤色）

面部与发型：

发型：发色、长度、发型（如：金色及肩卷发）。

面部特征：仅描述极度显著且客观的特征（如：蓄有络腮胡、戴圆形金属框眼镜、面颊有雀斑）。避免描述主观审美特征。

身材轮廓与体型：

整体轮廓：（如：高大魁梧、身材娇小、体型匀称）。

基于服装的合身度推断：（如：“穿着非常宽松的衣物，难以判断具体体型”）。

服装与配饰（客观清单式描述）：

上身：款式（T恤、衬衫、外套）、颜色（使用通用色名，如“靛蓝色”、“酒红色”）、图案、材质感（如：反光的皮质、粗糙的牛仔布）、合身度（紧身/合身/宽松）。

下身：款式（牛仔裤、半身裙）、颜色、图案、长度（如：九分裤、及膝裙）。

鞋袜：

鞋子：类型（运动鞋、高跟鞋）、颜色。

袜类（如可见）：类型（短袜、连裤袜）、颜色、透明度（透明/半透明/不透明）、长度（过膝/及膝/小腿）及任何可见纹理（如网眼、条纹），基于透肉程度，推断一个大概的丹尼尔数范围。。

配饰：列举可见配饰（如：黑色皮质腰带、银色腕表、帆布单肩包）。

精确姿态分析（关键部分）：

整体姿态与摄像机关系：

基础姿态：（如：坐姿、弓箭步、倚靠）。

躯干方向：（如：“躯干相对于地面倾斜约30度”）。

摄像机视角：（如：“从人物右侧方约45度进行平视拍摄”）。

肢体几何关系：

腿部：描述膝关节弯曲角度、大腿与小腿的相对位置、脚的支撑方式（全脚掌/脚尖）与指向。

手臂：

可见性声明：首先说明“左手/右臂是否被身体或其他物体遮挡”。

可见部分：描述肩、肘、腕关节的位置与角度，手的具体位置（如：“右手掌心压在地面上，位于右膝外侧10厘米处”）、手势（握拳、张开）及朝向。

头部与视线：

头部朝向：（如：“头部向左侧转动，与躯干形成扭动”）。

视线方向：（如：“视线向下，聚焦于手中拿着的书本”）。

三、 场景与环境
地点与空间：

主要场景：（如：现代办公室、凌乱的卧室、繁忙的城市十字路口、宁静的森林空地）。

关键物体与布局：列举并简要描述场景中重要的物体及其相对位置（如：“一张胡桃木书桌位于画面中央，桌上有一台打开的笔记本电脑和一堆散乱的文件”）。

光线与色彩：

光源：类型（自然光/人工光）、方向（侧光、顶光、逆光）与质量（硬光/强对比阴影、柔光/阴影柔和）。

色彩基调：整体色调（冷色调、暖色调）、饱和度高/低、有无突出的强调色。

阴影与高光：描述显著的阴影区域和高光区域。

构图与视角：

景别：（特写、中景、全景）。

镜头透视：（如：广角畸变、长焦压缩感）。

四、 整体氛围与潜在叙事线索（基于视觉元素的客观推断）
整体氛围：（如：紧张、悠闲、孤独、喧闹、科技感、复古感）。

互动与活动：基于人物姿态、表情和物体，描述图中正在发生的动作（如：“P1正在向P2展示手机屏幕上的内容”）。

图像风格与目的：（如：写实主义摄影、卡通风格插画、产品广告、社交媒体生活照）。
```

## 物品分析
### 丝袜广告

```
请你扮演一位专业的时尚分析师和纺织品质检员。请仔细分析用户提供的图片，重点关注图中人物穿着的丝袜，并按照以下结构和维度，输出一份极其详细、客观、视觉化的描述报告。这份报告将直接用于指导一个AI绘画模型进行精确还原。

请按以下框架进行分析和描述：

主体定位：

丝袜在图片中的穿着部位（如：覆盖双腿至腰部、仅小腿等）。

是否与其他服饰搭配（如：短裙、高跟鞋、吊袜带等），并描述衔接关系。

厚度与透明度（核心维度）：

视觉透肉程度： 描述皮肤细节（如血管、痣、膝盖/脚踝的骨骼轮廓、肤色本身）的可见程度。是“完全可见”、“朦胧可见”、“几乎不可见”还是“完全不可见”？

推断D数范围： 基于透肉程度，推断一个大概的丹尼尔数范围（如：5-15D， 30-50D， 80D以上等）。

质感形容： 看起来是“如皮肤般纤薄”、“轻薄微透”、“有实质感的哑光面”还是“厚重的针织感”？

颜色与光泽：

主色与色调： 精确描述颜色（如：冷调浅肤、暖调深肤、纯黑、炭灰、带紫调的深灰等）。

光泽类型： 是“完全哑光（无任何反光）”、“细腻的丝绸光泽”、“强烈的漆皮/亮光反光”，还是“带有细微闪粉/亮丝”？

光线互动： 描述光线在丝袜表面形成的效果（如：在腿部曲面形成一条平滑的高光带；在褶皱处产生明暗对比；整体呈现均匀漫反射等）。

纹理与材质细节：

表面纹理： 是“光滑无纹理”、“有细微的针织或天鹅绒纹理”、“有明显的网格/渔网花纹（请描述网格形状和大小）”，还是有“蕾丝、刺绣或拼接图案”？

特殊结构：

是否能看到从脚后跟延伸至大腿后侧的竖线骨线（后缝线）？

脚尖或脚后跟是否有加固加厚的区域？

腰部设计是怎样的？（如：宽边平面腰头、蕾丝宽边、有品牌logo等）。

是否为连裤袜、长筒袜、中筒袜或吊带袜？若是吊带袜，请描述吊袜带的样式和连接点。

穿着状态与动态细节（关键的真实感来源）：

合身度： 是“极度紧绷无褶皱”、“贴合但关节处有自然褶皱”，还是“略有松弛”？

褶皱特征： 请务必详细描述任何褶皱的位置、形状和成因。

膝盖处： 弯曲时形成的放射状或横向褶皱。

脚踝与脚背： 因鞋子或脚部弯曲产生的横向堆积褶皱或拉伸平滑区。

大腿/腰部： 因挤压或重力产生的横向褶皱或轻微下滑痕迹。

边缘处理： 长筒袜的袜口是平整、有罗纹，还是微微卷边？与皮肤边界是否清晰？

综合风格与氛围：

用几个关键词总结丝袜带来的整体感觉（如：专业优雅、性感魅惑、复古时尚、日常休闲、舞台戏剧化等）。

描述图片本身的拍摄风格（如：商业摄影棚光、自然光、暗调电影感、高调清新等），这会影响丝袜的呈现方式。

输出格式要求：
请将你的分析结果整理成清晰的条目或段落，确保描述是客观的、基于视觉事实的。最后，你可以将以上分析综合成一段 “用于AI绘画的提示词摘要” ，浓缩核心视觉特征，方便直接复制使用。
```
